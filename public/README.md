# Public Assets

Place your logo image here as `logo.png` for it to appear in the header and home page.

The logo should be:
- PNG format (or any web-compatible image format)
- Named exactly `logo.png`
- Recommended size: 200x200px or larger (will be scaled automatically)

The logo will be displayed:
- In the navbar (header) - scaled to 40x40px
- On the home page hero section - scaled to 128-192px depending on screen size
